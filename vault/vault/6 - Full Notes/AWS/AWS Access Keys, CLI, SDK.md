# AWS Access Keys, CLI, SDK

## How can users access AWS?
- AWS Management Console
- AWS Command Line Interface (CLI) - protected by access keys
- AWS Software Dev Kit (SDK) - for code: protected by AKs

Access keys are generated through AWS Console
Users manage their own access keys

## What's the AWS CLI
Tool to interact with AWS services using command line shell
- direct access to public APIs and you can develop scripts to manage resources
- open source!
- alternative to using AWS Management Console

## What's the AWS SDK
Set of libraries to use AWS in your program. kind of like jQuery
- AWS CLI is built on AWS SDK for Python
